# Proyecto_html
Proyecto html, contiene 5 paginas:
index: es la pagina principal, contiene navda y carousel de noticias, con imagenes donde cada imagen tiene
enlace para la seccion de noticias, tiene canción de fondo.
Multimedia: Contiene dos carousel uno de vídeos(2), donde el primer vídeo se reproduce 
solo al entrar a la página, ambos tomados de youtube y otro carousel de imagenes.
Historia: Contiene la breve historia junto con 3 imagenes, todas en diferentes posiciones.
Contacto: Contiene 7 campos, el disabled está en "Apellidos"
Noticias: Contiene 4 noticias.
Todas contienen favicon.

Proyecto_html contiene 4 ramas, la ultima "gh-pages" solo fue usada para obtener la dirección de nuestro sitio web,
por lo que no se hizo merge.
